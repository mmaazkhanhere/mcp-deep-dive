# OSI vs TCP/IP — Quick Compare

- **OSI (7 layers)**: Application, Presentation, Session, Transport, Network, Data Link, Physical.
- **TCP/IP (4–5 layers)**: Application, Transport, Internet, Link (Physical often grouped with Link).

## Remember

- OSI = conceptual teaching model.
- TCP/IP = practical Internet stack.
